## Overview

A desktop app to download YouTube subtitles. Built with Python and Qt.

## Features

- Download subtitles in multiple languages.
- Save as `.srt` or `.txt`.
- Lightweight, fast, and easy-to-use.

## Installation

### Prerequisites

- Python 3.x
- PyQt5
- youtube-dl (or `yt-dlp`)
- Requests
